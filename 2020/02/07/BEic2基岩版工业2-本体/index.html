BEic2基岩版工业2-本体

2020-02-07
作品 / BEic2

时隔两个多月,BIEC2也终于更新啦！还是感谢大家的一路支持qwq

基本信息:

中文名字:基岩版工业2
英文名字:BEic2
类型:Add-Ons
前置:MinecraftBE
最新版本:正式版V0.3.06
最新版本支持游戏版本:1.16.x
最后更新时间:2020-07-10
来源:原创

历史更新日志:

V0.3.06:

将自定义方块适配了1.16的版本，但是在物品栏里也找不到(API问题,作者不背锅)
只能像自定义物品一样,使用/give指令获得

本版本为紧急更新适配,并非V0.3.0全部内容

V0.2.0:

新增:

物品:

普通物品:
新增石粉
新增铀235,铀238,小撮铀235,小撮铀238,(合成核弹)
新增钚,小撮钚(同上)
新增能量水晶(合成辐射枪)
新增绝缘锡线(合成电路板)
新增粉碎铀矿石(合成小撮钚和小撮铀238)
新增树脂(获取橡胶)
新增没有电的电池与充满电的电池

工具:
新增工具 锤子(用于砸碎铀矿石制作粉碎铀矿石)
新增有电的挖矿/爆破辐射枪,没电的挖矿/爆破辐射枪(两种模式,挖矿模式和爆破模式,默认挖矿模式，潜行使用,用久后需要使用电池充电)
新增青铜头盔（半成品，直接用指令/replaceitem entity @p slot.armor.head 0 biyuehu:beic2_toukui_tong）

方块:

新增防爆玻璃(专门防核弹,抗暴堪比基岩)
新增铁炉(半成品)
新增铀块
新增没有燃料的核弹(没任何用)

实体:
新增工业tnt(威力是tnt4倍,并且一放就炸)
新增已填满燃料的核弹(范围最高可涉及到半径60的地方,130以内会有虚弱与剧毒buff)
辐射枪子弹(顾名思义)

配方:

新增铀235与铀238/小撮铀235与小撮铀238/钚与小撮钚之间的合成与分解(类似铁粒与铁锭之间的关系)

熔炼:

新增充满电的电池的熔炼方式:
高炉里:没有电的电池

新增小撮铀235的熔炼方式:
烟熏炉里:粉碎的铀矿石

新增小撮钚的熔炼方式:
高炉里:粉碎的铀矿石

新增橡胶的熔炼方式:
熔炉,烟熏炉,篝火里:树脂

其它:
新增全局资源中的作品信息GUI

修改&移除&修复bug:
将上个版本的电池为没有电的电池
修改了橡胶的获取方式
现在只能用绝缘锡线合成电路板
修改了物品”Biyuehu:BEic2_jian_tong”的lang文件名字

移除了上个版本获取1号彩蛋的function

修复了无法合成青金石粉和青金石板的bug

彩蛋:
